- 🛠️ Skills showcase
- 🚀 Projects gallery
- 📞 Contact section
- ⚡ Fast and user-friendly interface

1. Clone the repository:

```bash

```

2. Navigate to the project folder:

```bash
cd space-portfolio
```

3. Open `index.html` in your browser.
