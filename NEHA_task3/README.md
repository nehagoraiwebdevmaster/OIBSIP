# 🌡️ Temperature Converter

A simple and beginner-friendly Temperature Converter website that allows users to quickly convert temperatures between Celsius, Fahrenheit, and Kelvin.

## 🚀 Features

- Convert Celsius to Fahrenheit and Kelvin
- Convert Fahrenheit to Celsius and Kelvin
- Convert Kelvin to Celsius and Fahrenheit
- Simple and clean user interface
- Instant conversion results
- Responsive design for desktop and mobile devices

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

```

## 🎯 How It Works

1. Enter a temperature value.
2. Select the input temperature unit.
3. Choose the unit you want to convert to.
4. Click the **Convert** button.
5. View the converted temperature instantly.
